# iTerm2 Configuration

## Color Schemes
- `ben.itermcolors` - Custom color scheme
- `Snazzy.itermcolors` - Snazzy color scheme

Import via iTerm2 → Preferences → Profiles → Colors → Color Presets → Import

## Preferences
- `com.googlecode.iterm2.plist` - Full iTerm2 preferences export

### To Import Preferences
```bash
defaults import com.googlecode.iterm2 ~/path/to/dotfiles/iterm/com.googlecode.iterm2.plist
```

### To Update Preferences
```bash
defaults export com.googlecode.iterm2 ~/path/to/dotfiles/iterm/com.googlecode.iterm2.plist
```
