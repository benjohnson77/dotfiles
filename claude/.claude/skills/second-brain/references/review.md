# Review rituals — weekly review, commitment reconciliation, daily reflection

Capture is automated; *using* the vault is not. These rituals turn the pile into
foresight. Run on request ("weekly review", "what am I behind on") or on a cadence.

## Daily reflection (fills the blank Daily note)

The Daily note (`Daily/YYYY-MM-DD.md`) auto-fills "What's on today" but leaves
"What I'm thinking about / What happened / Decisions made / Tomorrow" empty. When Ben
debriefs a day, capture into those sections in his voice:

- **What happened** — the few things that actually mattered, linked to meetings.
- **Decisions made** — one line each; if material, spin out a `Decisions/` note
  (see synthesis.md) and link it.
- **What I'm thinking about** — open questions, worries, ideas → candidates for
  `Concepts/` or `Topics/`.
- **Tomorrow** — carry-forward priorities.

## Weekly review

1. **Pull the week**: `recent_activity` for the last 7 days + the week's Inbox digests.
2. **Commitments**: reconcile the digest's "Commitments I've made" — what closed, what
   slipped, what's newly past-due. Produce a tight "this week vs. still open vs. past
   due" summary. Flag anything past-due >2 weeks for a drop/delegate/do decision.
3. **Decisions**: scan the week's meetings for un-logged decisions (synthesis.md
   backfill loop). Write the confirmed ones.
4. **Clients/Initiatives**: for each active `Client`/`Initiative` touched this week,
   update "Status / current state" and the "History"/"Log" line. Retag stage/status
   if it changed.
5. **People**: enrich the 2–3 highest-value people Ben engaged with this week.
6. **Output**: a short review note (in `Daily/` or `Inbox/`) — wins, slips,
   decisions logged, and the top 3 priorities for next week. Link everything.

## Commitment reconciliation (standalone)

The digest already tracks commitments well. On demand, cross-check them against
reality: search recent meetings/Slack/Gmail context for evidence a commitment was
met, and mark `[x]` with a date. Surface stale commitments (past due, no movement) as
a decision for Ben: do / delegate / drop. Never silently close a commitment — confirm.

## Cadence

These can be scheduled (see the `schedule` skill / a routine). A good default: daily
reflection prompt in the evening, weekly review Friday afternoon or Monday morning.
Propose it; let Ben opt in rather than creating routines unprompted.
